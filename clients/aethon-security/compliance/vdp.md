# Vulnerability Disclosure Policy (VDP) — Aethon Security

**Documento:** VDP-AETHON-001
**Data:** 2026-06-11
**Classificação:** Público

---

## 1. Propósito

A Aethon Security compromete-se a manter os mais altos padrões de segurança. Este VDP
estabelece o processo para receção responsável de reportes de vulnerabilidades.

---

## 2. Escopo

- AethonShield v3.x (SIEM + endpoint agent)
- API pública do AethonShield
- Website e documentação oficial

---

## 3. Canais

- Email: `security@aethonsecurity.pt` (PGP: fingerprint disponível no website)
- Bug bounty: via plataforma [a designar]
- Resposta: 24h (critical), 72h (standard)

---

## 4. Processo

| Fase | Prazo |
|---|---|
| Confirmação de receção | 24h |
| Triagem | 72h |
| Investigação | 15 dias (critical: 48h) |
| Patch | 30 dias (critical: 7 dias) |
| Divulgação coordenada | 90 dias |

---

## 5. Safe Harbor

A Aethon Security compromete-se a:
- Não tomar ações legais contra boa-fé
- Não requerer dados além do necessário para demonstrar a vulnerabilidade
- Reconhecer publicamente o investigador (se autorizado)
